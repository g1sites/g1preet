# g1preet

Personal site for Jeevanpreet (G1) Singh — redesigned, static, mobile-first.

- `/` — homepage
- `/contact/` — JotForm contact form (https://form.jotform.com/g1preet/webcontact-g1p)
- `/help/` — QuickSupport (https://898.tv/g1help)

Stack: static HTML + Tailwind CDN + Fraunces/Inter (Bunny Fonts). No analytics, no trackers.

Deploy: any static host. `_headers` provides Cloudflare Pages / Netlify cache rules.
